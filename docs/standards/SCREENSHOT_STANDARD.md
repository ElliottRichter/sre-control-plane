# Screenshot Standards

Purpose:
Provide repeatable, professional documentation.

---

## Naming Convention

YYYY-MM-DD_system_feature_description.png

Examples:

2026-05-25_vm1_grafana_datasources.png
2026-05-25_vm2_jenkins_pipeline_success.png
2026-05-25_vm2_nagios_dashboard.png

---

## Rules

Always include:

- browser URL if non-sensitive
- timestamps when relevant
- entire UI panel
- visible status indicators

Never include:

- passwords
- API tokens
- IPs intended to remain private
- secrets
- SSH keys
- personal identifiers
